## a fashion fast wechat pay api for golang

## 时尚，简单，卓越，高性能的微信支付接口

获取包:
> go get github.com/godcong/wego

建议使用golang/dep添加包:  
> dep ensure --add github.com/godcong/wego

配置config.toml
> 参考config.toml.example
